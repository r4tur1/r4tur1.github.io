[X-Monitor](https://r4tur1.github.io/X-Monitor/)

A Python tool I created to monitor system and network activity. I wanted something lightweight that could track processes, log performance, or even alert me about unusual behavior. It’s still a work in progress, but the goal is to make it a customizable dashboard for keeping an eye on my machines. Great for learning about automation, data logging, and security.

[Bubble-Crossing](https://r4tur1.github.io/Bubble-Crossing/)

A fun little JavaScript game I made, inspired by endless runners like Crossy Road—but with a bubble-themed twist! The idea was to practice game mechanics, collision detection, and DOM manipulation while creating something playful. It’s not super polished yet, but I had a blast working on it, and I might add more levels or power-ups later.

[PokeDex](https://r4tur1.github.io/PokeDex)

As a Pokémon fan, I built this interactive Pokédex using the PokeAPI. It lets you search for Pokémon, see their stats, types, and abilities—basically a digital encyclopedia for trainers. I focused on making it clean, responsive, and easy to use. Working with APIs was a great learning experience, and I’m thinking of adding features like shiny forms or move-sets next.

[Pokinfo](https://r4tur1.github.io/Pokinfo)

This is my second Pokémon-related project, but with a deeper dive into data. While PokeDex is more visual, Pokinfo is about detailed analysis—maybe type matchups, evolution trees, or competitive stats. I’m still experimenting with it, but the goal is to make it a useful tool for players who love crunching numbers and strategizing.
